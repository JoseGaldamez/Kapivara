# Contributing to Kapivara

First off, thank you for considering contributing to Kapivara! We welcome valid contributions from the community to help improve this project.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your local machine.
3.  **Create a new branch** for your fix itself or feature (`git checkout -b feature/amazing-feature`).
4.  **Make your changes** and commit them with descriptive messages.
5.  **Push** to your fork.
6.  **Open a Pull Request** (PR) to the `main` branch of this repository.

## Development Workflow

- Please ensure your code works and does not break existing functionality.
- Try to follow the existing code style and structure.
- If you are adding a new feature, please describe it clearly in your PR.

## Reporting Bugs

If you find a bug, please create a GitHub Issue including:
- A clear description of the problem.
- Steps to reproduce.
- What you expected to happen vs. what actually happened.
