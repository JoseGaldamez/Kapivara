## ğŸ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   **Node.js** (Latest LTS recommended)
*   **Rust** (via `rustup`)
*   **Tauri CLI**

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/JoseGaldamez/Kapivara.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run tauri dev
    ```
